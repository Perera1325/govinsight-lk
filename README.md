\# GovInsightLK



GovInsightLK is a Sri Lanka Government Intelligence Platform that analyzes ministry changes, budget allocations, and institutional restructuring.



\## Features



\- Government structure tracking

\- Budget allocation analysis

\- Ministry change impact insights

\- Interactive dashboard

\- Data-driven governance intelligence



\## Tech Stack



Backend:

\- FastAPI

\- PostgreSQL

\- Pandas



Frontend:

\- React

\- Chart.js



Data Sources:

\- OpenData.lk

\- Government Gazette



\## Status



🚧 In Development (Sprint Phase)



\## EquiLanka Poverty Intelligence Module



EquiLanka is a smart poverty analysis and resource allocation prototype designed to help identify vulnerable households and recommend targeted government interventions.



\### Features



\- Household poverty scoring

\- Automated recommendations (housing, sanitation, education, financial aid)

\- Community-level poverty analytics

\- Interactive dashboard visualization

\- Chart-based insights



\### API Endpoints



POST /poverty/analyze  

POST /poverty/community



\### Purpose



This system aims to support data-driven policy decisions to reduce poverty and improve living conditions in Sri Lanka.
